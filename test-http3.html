📖 如何启用HTTP/3

Chrome浏览器:

  1. 在地址栏输入: chrome://flags/#enable-quic
  2. 将 "Experimental QUIC protocol" 设置为 Enabled
  3. 重启浏览器
  4. 再次访问此页面查看是否使用HTTP/3

Firefox浏览器:

  1. 在地址栏输入: about:config
  2. 搜索: network.http.http3.enabled
  3. 设置为 true
  4. 刷新此页面

Edge浏览器:

  1. 在地址栏输入: edge://flags/#enable-quic
  2. 将 "Experimental QUIC protocol" 设置为 Enabled
  3. 重启浏览器

🛠️ 开发者工具检测

按 F12 打开开发者工具:

  1. 切换到 Network (网络) 标签
  2. 刷新页面 (F5)
  3. 点击任意请求，查看 Protocol (协议) 列
  4. 如果显示 h3 或 http/3，说明HTTP/3已启用！
